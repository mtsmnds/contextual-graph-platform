# PRD0016 — TipTap UI Phase 2: Free Notion-like Delta

**Date:** 2026-05-15
**Status:** Complete
**Milestone:** M2 — Reading Workspace

---

## Context

Phase 1 (PRD0014) delivered the Simple Editor scaffold with a full toolbar. The editing experience was functional but lacked two hallmark features of block editors: a floating toolbar on text selection (BubbleMenu) and block drag-and-drop reordering (Drag Handle). Additional free extensions (Placeholder, Emoji) were planned but not yet wired.

---

## Decision

Add the following free (MIT) features to the editor:

1. **BubbleMenu** — floating toolbar that appears when text is selected, providing quick access to bold, italic, strike, underline, code, highlight, and link
2. **Drag Handle** — six-dot grip icon on block hover, enabling drag-and-drop reordering of paragraphs, headings, lists, and other blocks
3. **Placeholder** — "Start writing..." hint text in empty editor
4. **Emoji** — `:smile:` → 😄 autocomplete using `@tiptap/extension-emoji`

All features are free, no paid subscription required.

---

## Implementation

- **New packages:** `@tiptap/extension-placeholder`, `@tiptap/extension-emoji`, `@tiptap/extension-drag-handle-react`, `@tiptap/extension-drag-handle`, `@tiptap/extension-node-range`
- **BubbleMenu** imported from `@tiptap/react/menus`, rendered inside `EditorContext.Provider`
- **DragHandle** wraps `EditorContent`, uses simple SVG grip icon
- **Placeholder** and **Emoji** added to extensions array in `TiptapEditor`
- Existing extensions (Underline, Highlight, TextAlign, TaskList, Typography, Superscript, Subscript) were already included by the Simple Editor template and require no additional packages
